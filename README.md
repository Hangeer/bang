#### 重邮帮小程序开发脚手架

#### 使用方法
+   cnpm install 安装相关依赖
+   npm run watch 开启监视 刷新
+   npm run generate 生成一个新的 page (并且在顶层的 app.json 中添加这个 page) 有命令行交互
+   npm run build 
+   开发的时候小程序项目文件夹选中 dist 文件夹就 OK
